Installation

Install on PyPi

You can install RadTraQ using the following:

pip install radtraq

Required Dependencies

RadTraQ requires the following software.

  • Python >=3.7.x

  • NumPy

  • SciPy

  • Xarray

  • Cython

  • matplotlib

  • netCDF4

  • ACT

  • pint
