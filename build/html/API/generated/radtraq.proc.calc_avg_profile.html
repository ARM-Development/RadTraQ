radtraq.proc.calc_avg_profile¶

radtraq.proc.calc_avg_profile(_obj, variable=None, mask='mask2', mask_val=1, first_height=500.0, height='range', ygrid=None)[source]¶

Function for calculating average profiles from data after applying the cloud mask

Parameters:
  • _obj (xarray Dataset) – xarray object with all the data
  • variable (list) – List of variables to average
  • mask (string) – Name of mask in file to apply to data
  • mask_val (int) – Value of mask to threshold on
  • first_height (float) – First height to start the analysis. Start at 500 to throw out non-meteorological echo
  • height (string) – Name of the height variable to use
  • ygrid (numpy array or list) – Numpy array of ygrid values to interpolate to
Returns:

_obj – xarray dataset with new variables added
Return type:

xarray Dataset
